# How-to-do-DC-Motor-Speed-Control-with-android-phone
This application we made with Flutter-Dart codes is a DC Motor speed control application.  Using the Slider widget, we generate analog values between 0 and 250 and  send this value to the bluetooth adapter with the sendData function.
